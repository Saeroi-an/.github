# 새로이 안(安)

당신의 편안한 진료를 위해, 사용자 맞춤 AI 의료 서비스 

---

## 📝 소개 (Overview)

새로이 + 편안할 안(安)

한국에 체류하고 있는 외국인을 위한 의료서비스입니다. 
더 이상 어렵게 한국어로 이야기하지 않아도 돼요. 
새로이 안으로 간편하게 본인의 증상을 전달해보세요!
사용자 정보를 기반으로 처방전 인식, 셀프 진단 체크, 주변 정보를 제공받습니다. 

---

## 📝 공모전 (Overview)

![프로젝트 미리보기](./공모전.png)

'인공지능(AI), 충북의 미래를 디자인하다'
전국 ICT 융합 공모전을 함께 준비하고있습니다. 

---

## 🚀 주요 기능 (Features)
- 사용자 정보 입력 및 관리
- AI 기반 처방전 인식 및 채팅
- 셀프 진단 체크
- 다국어 지원 (한국어 / 중국어 / 영어)
- React Native + FastAPI 연동 구조
